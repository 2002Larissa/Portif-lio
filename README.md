<h1>🚀 Meu Portfólio de Desenvolvedora Back-end<h1>
<p>Este é o meu portfólio pessoal, projetado para apresentar meus projetos, habilidades técnicas e facilitar o contato para oportunidades de colaboração no ecossistema Back-end.</p>

<h2>🎨 Sobre o Design<h2>
O projeto foi construído com uma estética Deep Blue, focada em uma experiência imersiva e moderna.

Seções Full-Screen: Navegação fluida onde cada seção ocupa 100% da tela do usuário.

Fundo Animado: Utilização da biblioteca Particles.js para criar uma rede de pontos interligados que reagem ao movimento do mouse, simbolizando conexões de rede e sistemas.

Glassmorphism: Cards de projetos e habilidades com efeito de vidro fosco (backdrop-filter) para profundidade visual.

Responsividade: Layout adaptável para dispositivos móveis, tablets e desktops utilizando CSS Grid e Flexbox.

🛠️ Tecnologias Utilizadas
HTML5: Estrutura semântica da página.

CSS3: Estilização avançada, animações de entrada e layout responsivo.

JavaScript (Vanilla): Lógica de rolagem suave e interatividade.

Particles.js: Biblioteca externa para o efeito de partículas no background.
